# House Price Prediction Using Machine Learning

## Overview

This project predicts house prices based on property area using Machine Learning. A Linear Regression model is trained on housing data to estimate the price of a house accurately.

## Features

* Data preprocessing using Pandas
* Linear Regression model training
* House price prediction based on area
* Simple and beginner-friendly implementation
* Easy to understand and modify

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib

## Project Structure

House-Price-Prediction/

├── house_price.py

├── house_data.csv

├── requirements.txt

└── README.md

## Installation

1. Clone the repository:
   git clone https:https://github.com/yourusername/House-Price-Prediction.git

2. Navigate to the project folder:
   cd House-Price-Prediction

3. Install required libraries:
   pip install -r requirements.txt

## Usage

Run the Python file:

python house_price.py

Enter the house area when prompted to get the predicted price.
